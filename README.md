## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ashish0719/Farmora.git
```

---

### Frontend Setup

```bash
cd client

npm install

npm run dev
```

Frontend runs on:

```text
http://localhost:3000
```

---

### Backend Setup

```bash
cd strapi-server

npm install

npm run develop
```

Backend runs on:

```text
http://localhost:1337
```

---

## 🔑 Environment Variables

### Frontend (.env.local)

```env
NEXT_PUBLIC_STRAPI_URL=http://localhost:1337
```

### Backend (.env)

Configure Strapi environment variables according to your local setup.
